Personnalisation clé-en-main et ressources Siiimple.

== Description ==
L'extension Siiimple fournit une solution clé-en-main dédiée, simplifie l'interface d'administration WordPress et permet d'accéder facilement aux ressources de l'agence de marketing Siiimple.
